# แบบฝึกหัดอัลกอริทึม

โปรเจกต์นี้ประกอบด้วยแบบฝึกหัดอัลกอริทึมในภาษา JavaScript

## แบบฝึกหัด

### แบบฝึกหัดที่ 1: ย้ายเลขศูนย์ (`ex-1.js`)
ย้ายเลขศูนย์ทั้งหมดไปไว้ท้ายอาร์เรย์ โดยรักษาลำดับของตัวเลขที่ไม่ใช่ศูนย์ไว้

**ตัวอย่าง:**
```javascript
moveZeroes([0, 1, 0, 3, 12]) // คืนค่า [1, 3, 12, 0, 0]
moveZeroes([0]) // คืนค่า [0]
```

### แบบฝึกหัดที่ 2: แปลงเลขโรมันเป็นจำนวนเต็ม (`ex-2.js`)
แปลงสตริงเลขโรมันเป็นจำนวนเต็ม

**ตัวอย่าง:**
```javascript
romanToInt("III") // คืนค่า 3
romanToInt("LVIII") // คืนค่า 58
romanToInt("MCMXCIV") // คืนค่า 1994
```

## วิธีการใช้งาน

```bash
# ติดตั้ง dependencies (ถ้ามี)
npm install

# รันแบบฝึกหัดที่ 1
npm run ex1

# รันแบบฝึกหัดที่ 2
npm run ex2

# หรือรันโดยตรงด้วย node
node ex-1.js
node ex-2.js
```

## โครงสร้างโปรเจกต์

```

├── ex-2.js          # แบบฝึกหัดแปลงเลขโรมัน
└── README.md        # ไฟล์นี้
```
